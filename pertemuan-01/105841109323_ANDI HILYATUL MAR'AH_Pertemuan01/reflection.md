Refleksi dan Harapan
Melalui praktikum ini, saya menyadari bahwa menyiapkan infrastruktur yang tepat adalah langkah awal yang paling menentukan dalam siklus hidup pengembangan perangkat lunak modern.
Harapan saya dari praktikum ini adalah:
- Dapat memahami alur kerja kontainerisasi menggunakan Docker secara mendalam agar bisa mendeploy aplikasi dengan standar industri.
- Mampu mengimplementasikan otomatisasi sederhana untuk meningkatkan efisiensi pengembangan kode.
- Membangun kebiasaan kerja yang rapi dan kolaboratif dengan memanfaatkan fitur Git dan GitHub secara optimal.
